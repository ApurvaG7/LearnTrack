# ⚙️ Environment Setup & JVM Understanding
JDK Version Used : JDK 21

Installation Method : Installed automatically through IntelliJ IDEA

IntelliJ configured the JDK and JVM internally (no manual setup required)


* Hello World Program Execution

1. Installed IntelliJ IDEA 
2. Created a new Java project 
3. Selected JDK 21 during project setup 
4. Created a HelloWorld class 
5. Ran the program using IntelliJ’s Run option

    public class Main {
